## 🎯 About The Project

**Grocify** is a modern, responsive grocery delivery website that showcases a full-featured landing page for an on-demand grocery service. The platform promises delivery in **under 15 minutes**, connecting customers with fresh, farm-sourced produce and daily essentials.

This project was built as part of a web development portfolio to demonstrate frontend design skills including responsive layouts, interactive UI components, product filtering, cart management, and clean UX practices.

---

## ✨ Features

- ⚡ **Hero Section** — Eye-catching banner with CTA buttons and delivery stats
- 🧾 **Product Listings** — Dynamic product cards with category filtering (Fruits, Vegetables, Dairy, Bakery)
- 🛒 **Add to Cart** — Interactive cart with item count badge
- ❤️ **Wishlist** — Add/remove products to wishlist
- 📦 **Services Section** — Highlights fast delivery, local store partnerships, and mobile app
- 💬 **Contact Form** — Fully structured contact form with subject selection
- 📧 **Newsletter Subscription** — Email subscription component in the footer
- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop screens
- 🌿 **Eco-Friendly Theme** — Sustainable & organic brand identity
- 🔢 **Stats Counter** — Animated counters for customers, products, and delivery time

## 🚀 Getting Started

Follow these steps to run the project locally:

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge)
- [VS Code](https://code.visualstudio.com/) (recommended)
- [Live Server Extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) (optional but recommended)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Daniish-Qureshi/grocify-website.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd grocify-website
   ```

3. **Open in browser**
   - Simply open `index.html` in your browser, **OR**
   - Use Live Server in VS Code for hot reload:
     - Right-click `index.html` → **Open with Live Server**

That's it! No npm install, no build step needed. 🎉
